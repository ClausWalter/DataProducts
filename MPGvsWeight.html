2D Plots - MPG vs. Weight:

This is a 2D scatter diagram, which shows for the various cars analyzed the relation between MPG vs. Weight. The colors display the number of cylinders, the size of the data points gives an impression of the horse powers per data point. Each data point symbolizes a car in the underlying data set. You can hover over any data point, displaying the according values. If you click on any, the value will be memorized in one of the text fields beneath the plots. The second text field displays the value the mouse cursor is currently hovering over. This feature migh be useful to compare two values in the plot.
